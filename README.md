# العربية

## نظرة عامة

هذا المشروع عبارة عن **منصة احترافية لعرض الخدمات ومعرض أعمال للعملاء**، تم تصميمها لتقديم تجربة بصرية حديثة وسلسة لعرض التغطيات، الأعمال، والخدمات بشكل منظم وجذاب.

يجمع الموقع بين:

- **واجهة عرض أنيقة للزوار**
- **لوحة تحكم قوية وسهلة الاستخدام** لإدارة المحتوى بالكامل

ويُعد مناسبًا لـ:

- المصورين
- مقدمي التغطيات
- منظمي المناسبات
- أصحاب المعارض الرقمية
- مقدمي الخدمات الإبداعية

---

## فكرة المشروع

يعتمد المشروع على 3 محاور رئيسية:

### 1) عرض الخدمات
يتم عرض الخدمات بأسلوب بصري احترافي مع **عناوين متحركة وعبارات جذابة** لرفع التفاعل وإبراز هوية الموقع.

### 2) معرض أعمال للعملاء
يوفر الموقع **معرضًا منظمًا للأعمال والصور والتغطيات** بحيث يمكن للزائر تصفح الأعمال بسهولة وسرعة.

### 3) لوحة تحكم قوية وسهلة
يحتوي المشروع على **لوحة تحكم مرنة وسهلة الاستخدام** لإدارة:

- الأقسام الرئيسية
- التغطيات
- الصور
- المحتوى
- التعديل والحذف والإضافة

بالإضافة إلى دعم **Drag & Drop** لتسهيل ترتيب العناصر والصور بشكل عملي وسريع.

## تحسين الصور

يدعم المشروع تحسين الصور لرفع أداء الموقع وتحسين تجربة المستخدم من خلال:

### ضغط الصور
يتم ضغط الصور لتقليل حجم الملفات وتسريع تحميل الصفحات.

### التحويل إلى WebP
يتم تحويل الصور إلى صيغة **WebP** الحديثة لما توفره من:

- حجم أقل
- سرعة تحميل أفضل
- أداء أعلى
- تحسين تجربة المستخدم

# English

## Overview

This project is a **professional platform for showcasing services and client portfolios**, designed to deliver a modern and smooth visual experience for presenting services, event coverages, and creative work in an organized and attractive way.

The platform combines:

- A **clean and elegant public-facing interface**
- A **powerful and easy-to-use admin dashboard** for full content management

It is suitable for:

- Photographers
- Event coverage providers
- Wedding organizers
- Creative professionals
- Digital portfolio owners

---

## Project Idea

The project is built around 3 main pillars:

### 1) Service Showcase
Services are presented in a visually appealing way using **animated headings and engaging phrases** to capture visitor attention and strengthen the brand identity.

### 2) Client Portfolio Gallery
The platform provides a **well-organized gallery for projects, images, and coverages**, making it easy for visitors to browse previous work.

### 3) Powerful & Easy Admin Dashboard
The system includes a **flexible and easy-to-use admin panel** to manage:

- Main sections
- Coverages
- Images
- Content
- Add / Edit / Delete operations

It also supports **Drag & Drop** functionality for easier sorting and organizing of items and images.

---

## Features

- Professional service presentation
- Organized portfolio gallery
- Powerful and easy admin dashboard
- Main and sub-section management
- Image and coverage management
- Animated headings with attractive phrases
- **Drag & Drop** support
- Responsive design for all devices
- Clean and modern user experience
- Performance-focused structure

## Image Optimization

The project includes image optimization features to improve performance and loading speed:

### Image Compression
Images are compressed to reduce file size and speed up page loading.

### WebP Conversion
Images are converted to **WebP** format for:

- Smaller file sizes
- Faster loading
- Better performance
- Improved user experience
- Better SEO optimization
